Text Formatting Tags

There are different text formatting tags we can use in HTML. Like the BOLD (b) tag and the Italic (I) tag for font. We use the Emphasis(em) tag for emphasis. There is also the Quote(q) tag for quotes like An apple a day keeps the doctor away. If we have brought section from another source than we use the Blockquote(blockquote) tag. For example:

The Laliguras flower, also known as the Rhododendron flower, proudly represents Nepal as its national flower.
Then there is the Deleted(del) tag which strikethrough the text often accompanied by Inserted(ins) tag. Like:The banquet is at 6:00pm 7:00pm Hello
